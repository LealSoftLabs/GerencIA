# Frontend GerencIA Atualizado

Esta versão atualiza o frontend com:
- Agenda corrigida sem cards vazando.
- Tela de Profissionais mais completa, com barras, metas e informações.
- Tela Financeira com gráfico maior estilo Claude, mas mantendo o visual premium.
- Tela de Comissões mais completa.
- Tela de Marketing IA com calendário visual.
- Tela de Relatórios mais completa, com filtros, heatmap e retenção.
- Responsividade melhorada.

## Como usar

Você pode substituir a pasta `frontend` antiga por esta, ou copiar apenas:
- `src/main.jsx`
- `src/styles.css`

Depois rode:

```bash
npm install
npm run dev
```
